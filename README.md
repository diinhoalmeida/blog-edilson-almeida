## 👨‍💻 Rodando a Aplicação

- Clonando o projeto:
```
$git clone https://github.com/diinhoalmeida/blog-edilson-almeida.git
```

- Checando pastas no diretório em questão: 
```
$ls
```

- Entrando na pasta do projeto:
```
$cd market-place-ct-front
```

- Istalando depêndencias:
```
$npm i
```

- Iniciando o projeto:
```
$npm run dev
```
